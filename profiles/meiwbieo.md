## Création
```markdown
# **Concept & Messages – [Nom Enseigne]**
*Stratégie orientée trafic, panier moyen et image prix*

---

## **1. Insights Clés (Data-Driven)**
*Basés sur les données magasin et tendances secteur*

- **Trafic en baisse les mardis/jeudis** (-18% vs moyenne hebdo) → *Opportunité : créer un rendez-vous "mid-week"* avec une mécanique promotionnelle récurrente (ex: "Le Mardi Malin").
- **Panier moyen inférieur de 12% vs concurrents directs** → *Levier : bundles "prêts à cuisiner" (ex: "1 plat + 1 accompagnement + 1 dessert = -20%")* pour inciter à l’achat groupé.
- **Catégorie "Snacking" en croissance (+23% CA)** mais sous-représentée en linéaire → *Priorité : visibilité accrue (îlots dédiés, signalétique "Idées Repas Express")*.
- **Clients fidèles (carte enseigne) dépensent +35%** mais seulement 28% des visiteurs l’utilisent → *Urgence : mécaniques d’adhésion simplifiées (ex: "Scan & Go" avec réduction immédiate)*.
- **Perte de parts de marché sur les MDD premium** (-8%) → *Risque : image "low-cost" qui se renforce. Contre-attaque : storytelling "Qualité Accessible" (ex: "Nos producteurs locaux à prix juste")*.

---

## **2. Concept Principal : "L’Épicerie qui Vous Facilite la Vie"**
*Positionnement : **Gain de temps + Maîtrise du budget** (vs "juste des promos")*

### **A. Mécanique Phare : "Le Menu Express"**
**Objectif** : Augmenter le panier moyen (+15%) et la fréquence de visite (+10%).
**Dispositif** :
- **Offre** : 1 plat principal + 1 accompagnement + 1 produit snacking = **-20%** (vs achat à l’unité).
  *Exemple* : "Pâtes carbonara + salade César + cookie = 4,99€" (vs 6,50€).
- **Activation** :
  - **En magasin** : Îlot dédié en tête de gondole avec visuels "Repas en 10 min".
  - **Digital** : Push notification le matin ("Votre menu du soir est prêt !") + géolocalisation (offre déclenchée à proximité du magasin).
  - **Fidélisation** : Carte enseigne = menu offert après 5 achats.
**Mesure** :
  - Taux de conversion sur l’îlot (objectif : 25% des clients).
  - Panier moyen des acheteurs du bundle (cible : +20% vs moyenne).

### **B. Opération "Le Mardi Malin"**
**Objectif** : Lisser le trafic (-18% les mardis/jeudis) et écouler stocks.
**Dispositif** :
- **Promo ciblée** : **-30% sur les dates courtes** (produits frais à J-2/J-1) + **-15% sur les MDD** pour les détenteurs de la carte.
- **Communication** :
  - Affichage magasin : "Aujourd’hui, on vide vos frigos !" avec pictos produits concernés.
  - Réseaux sociaux : Stories "Chasse aux bonnes affaires" (ex: "Trouvez le yaourt à 0,20€ en rayon").
**Mesure** :
  - Trafic le mardi (cible : +20% vs moyenne).
  - Réduction du gaspillage (objectif : -15%).

### **C. Storytelling "Qualité Accessible"**
**Objectif** : Redorer l’image prix sans renier la qualité.
**Dispositif** :
- **Campagne "Nos Producteurs ont un Visage"** :
  - **En magasin** : Portraits de producteurs locaux sur les linéaires MDD premium + QR code vers vidéos "leur savoir-faire".
  - **Digital** : Série de posts "Le vrai coût de la qualité" (ex: "Pourquoi nos tomates coûtent 0,10€ de plus ?").
- **Preuve sociale** : Avis clients mis en avant ("Je fais mes courses ici sans culpabiliser").
**Mesure** :
  - Enquête image post-campagne (cible : +15 pts sur "qualité perçue").
  - Ventes MDD premium (objectif : +10%).

---

## **3. Recommandations Complémentaires**
*Actions rapides, faible coût, haut impact*

| **Levier**          | **Action**                                                                 | **KPI**                          |
|---------------------|---------------------------------------------------------------------------|----------------------------------|
| **Signalétique**    | Remplacer les étiquettes promos par des messages bénéfices clients :      | Taux de lecture (eye-tracking)   |
|                     | *"Ce pack de 6 yaourts = 2 petits-déj offerts"* vs *"-20%"*               |                                  |
| **Checkouts**       | Ajouter des *micro-offres* en caisse :                                    | Panier moyen (+5%)               |
|                     | *"1 barre de céréales achetée = 1 café offert"* (partenariat avec machine à café). |                                  |
| **Drive**           | Créer un onglet *"Repas de la semaine"* avec listes pré-remplies         | Taux de conversion drive (+12%)  |
|                     | (ex: "Menu étudiant", "Dîner en famille").                                |                                  |
| **Réseaux Sociaux** | Lancer un challenge *"Mon panier à 10€"* (clients postent leurs achats    | Engagement (+30%)               |
|                     | malins avec #NomEnseigne).                                                |                                  |

---

## **4. Calendrier & Budget Prévisionnel**
*Priorisation sur 3 mois*

| **Action**               | **Timing**       | **Budget**       | **Responsable**       |
|--------------------------|------------------|------------------|-----------------------|
| Lancement "Menu Express" | Semaine 1        | 15K€ (ILV + digital) | Chef de rayon Frais   |
| Opération "Mardi Malin"  | Semaine 3 (test) | 8K€ (affichage + promo) | Responsable Promo |
| Campagne "Producteurs"   | Mois 2           | 20K€ (production vidéo + RP) | Chef de marque MDD |
| Optimisation checkouts  | Continu          | 2K€ (étiquettes) | Responsable Magasin |

---
```
