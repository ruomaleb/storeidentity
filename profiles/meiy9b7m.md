# **Plan Média Local – [Nom Enseigne]**
*[Nom Magasin] – [Ville]*
**Objectif** : Maximiser l’efficacité incrémentale et le *reach utile* (ciblage des clients potentiels à fort ROI) via une approche data-driven et locale.

---

## **1. Insights Clés (Data-Driven)**
### **Profil Client & Zone de Chalandise**
- **Démographie dominante** :
  - [X]% des clients sont des [tranche d’âge, ex: 25-45 ans] (source: données CRM).
  - [Y]% de ménages avec enfants (vs [moyenne nationale/locale]), opportunité pour les produits [ex: familial, bio, etc.].
  - *Insight* : Prioriser les médias consommés par cette cible (ex: réseaux sociaux pour les 25-35 ans, presse locale pour les 50+).

- **Comportement d’achat** :
  - Pic de fréquentation les [jours/heures, ex: samedis 10h-12h] (source: données trafic).
  - Panier moyen de [€X] (vs [€Y] pour le réseau), avec surreprésentation des catégories [ex: frais, drive].
  - *Insight* : Synchroniser les campagnes avec les pics de trafic et promouvoir les catégories à marge élevée.

- **Concurrence locale** :
  - [Nom Concurrent 1] à [distance] km avec une part de voix média de [X]% (source: audit terrain).
  - [Nom Concurrent 2] utilise fortement [canal, ex: affiches 4x3], mais faible présence digitale.
  - *Insight* : Occuper l’espace digital sous-exploité et différencier via des offres locales (ex: partenariats avec producteurs régionaux).

- **Sensibilité aux promotions** :
  - [Z]% des ventes liées aux promos (source: data caisses), avec un taux de conversion de [A]% sur les flyers distribués en magasin.
  - *Insight* : Optimiser le mix promo/média pour éviter la cannibalisation (ex: limiter les flyers si saturation).

| Booster le panier moyen | +[Y]€/panier (catégories cibles) | Emailing CRM + PLV dynamique |
| Fidéliser les clients locaux | +[Z]% de fréquence d’achat   | Programme de parrainage + réseaux sociaux |

### **Mix Média Recommandé**
**Budget total estimé** : [€XX,XXX] (réparti comme suit) – *ROI attendu* : [X]:1.

| **Canal**               | **Allocation** | **Actions Clés**                                                                 | **Mesure d’Efficacité**                     |
|-------------------------|----------------|----------------------------------------------------------------------------------|---------------------------------------------|
| **Digital Local**       | 40%            | - Campagnes Facebook/Instagram géolocalisées ([rayon] km autour du magasin).     | Taux d’engagement (>[X]%), trafic en magasin via coupons digitaux. |
|                         |                | - Google Ads (recherche locale + display) ciblant "[enseigne] [ville]".         | CTR >[Y]%, conversions "Itinéraire" ou "Click-to-Call". |
|                         |                | - Partenariat avec [plateforme locale, ex: "Too Good To Go"] pour visibilité.    | Nombre de nouveaux clients acquis.          |
| **Affichage Extérieur** | 25%            | - 2 affiches 4x3 sur axes à fort trafic (ex: [rue/arrêt de bus spécifique]).     | Étude de notoriété post-campagne (sondage en magasin). |
|                         |                | - Panneaux digitaux en gare/centre commercial (si budget).                      | Trafic incrémental les jours suivant l’affichage. |
| **Presse Locale**       | 15%            | - Insertions dans [Journal Local] (cibler les jours de sortie des flyers promo). | Coupons découpés (trackés via code unique). |
| **CRM & Emailing**      | 10%            | - Emailing personnalisé aux clients inactifs (>30 jours) avec offre exclusive.  | Taux d’ouverture (>[Z]%), taux de conversion. |
| **Événementiel Local**  | 10%            | - Animation en magasin avec [partenaire local, ex: dégustation produit régional].| Nombre de participants, ventes incrémentales le jour J. |

---
## **3. Calendrier & Phasage**
### **Périodes Clés**
- **Lancement (Semaine 1-2)** :
  - Focus sur la notoriété : affichage + digital (vidéos "découverte du magasin").
  - *Budget* : 30% du total.
- **Promotion (Semaine 3-6)** :
  - Synchronisation avec les flyers promo (ex: "3 jours spéciaux").
  - *Budget* : 50% du total (pic sur les canaux convertisseurs : Google Ads + emailing).
- **Fidélisation (Semaine 7+)** :
  - Programme de parrainage ("1 achat = 1 cadeau pour un ami").
  - *Budget* : 20% du total.

### **Exemple de Planning Hebdomadaire**
| Semaine | Action                                  | Canal Principal          | Budget   |
|---------|-----------------------------------------|---------------------------|----------|
| 1       | Lancement campagne "Découvrez notre magasin" | Affichage + Facebook      | €X,XXX   |
| 3       | Promotion "Panier Malin" (catégories cibles) | Flyer + Google Ads        | €Y,YYY   |
| 5       | Événement "Producteurs Locaux"          | Événementiel + Instagram  | €Z,ZZZ   |

---
## **4. Recommandations Opérationnelles**
### **Optimisations Incrémentales**
- **A/B Testing** :
  - Tester 2 visuels Facebook (1 "promo" vs 1 "émotionnel") et allouer 80% du budget au gagnant après 7 jours.
  - *Outils* : Meta Ads Manager, Google Optimize.
- **Géomarketing** :
  - Exclure les zones déjà couvertes par [Concurrent 1] dans le ciblage digital pour éviter le gaspillage.
  - *Outils* : Google My Business, données mobiles (si disponibles).
- **Synergie Online/Offline** :
  - Intégrer des QR codes sur les affiches renoyant vers une landing page avec coupon (trackable via Google Analytics).
  - Former les équipes en magasin à mentionner les promos digitales ("Montrez ce code en caisse !").

### **Mesure & Reporting**
- **Tableau de Bord Hebdomadaire** :
  | Métrique               | Source                  | Cible   |
  |------------------------|-------------------------|---------|
  | Trafic en magasin      | Compteurs physiques     | +[X]%   |
  | Ventes incrémentales   | Data caisses (vs N-1)   | +[Y]€   |
  | Coupons digitaux utilisés | CRM/Google Analytics  | [Z]%    |

- **Outils Recommandés** :
  - **Tracking** : Google Analytics 4 + CRM enseigne.
  - **Attribution** : Modèle "last touch" pour les canaux digitaux, sondages en magasin pour l’affichage.

---
## **5. Risques & Contingences**
| Risque                          | Solution de Contournement                          |
|---------------------------------|----------------------------------------------------|
| Saturation publicitaire locale  | Limiter à 2 canaux principaux (ex: digital + 1 média offline). |
| Budget dépassé sur le digital   | Plafonner les enchères Google/Facebook à [€X]/clic. |
| Faible engagement sur les RS    | Pivoter vers du contenu UGC (ex: clients ambassadeurs). |
